🧠 How It Works
Observer Pattern

When you add a transaction, FraudAnalysisService automatically notifies all registered FraudDetector observers, re-running every algorithm on the updated graph.

Strategy Pattern

Each detector (Cycle, Hub, RapidHop) implements the same FraudDetector interface. The service iterates through all strategies without knowing their internals.

BFS Propagation

After detection, PropagationScorer runs BFS from every fraud node, computing risk scores with exponential decay: 100 / 2^hop per hop distance.
